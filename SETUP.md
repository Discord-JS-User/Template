## Setting Up

-   Fill `README.md`
-   Setup Docs Text Replacements in `typedoc.json`

-   Write **Code** In `src`
-   Write **Tests** In `test`

## Readying For Publish

#### `npm run ready`

-   Re-installs and Updates Packages
-   Prettifies all Files
-   Builds Package (in `dist` folder)
-   Generates Documentation (in `docs` folder)
